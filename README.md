## 📖 简介

本仓库用于存储 [**Angluswang**](https://blog.angluswang.com) 个人博客的评论数据。  
评论系统基于 [giscus](https://giscus.app/) 搭建，评论内容以 GitHub Discussions 的形式管理。

- 🛡️ 安全可靠
- 🗂️ 结构清晰
- 🔥 实时同步

---

## 🔗 快速访问

- 🚀 **个人博客**：[https://blog.angluswang.com](https://blog.angluswang.com)
- 🛠️ **giscus 评论系统**：[https://giscus.app/](https://giscus.app/)
- 📚 **本仓库 Discussions**：[Discussions 页面](../../discussions)

---

## ⚙️ 使用说明

1. 访问任意博客文章页面。
2. 滑动到评论区域，使用 GitHub 账号登录。
3. 直接发表评论、点赞、回复。
4. 评论将同步到本仓库的 Discussions 中存档。

---

## 🛡️ 注意事项

- 评论前请遵守基本的社交礼仪。
- 禁止发布违法、不当言论，违者责任自负。
- 管理员有权删除不符合规范的评论。
